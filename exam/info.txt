    div.style.height = HEIGHT + "px";

    window.onresize = function () {
        WIDTH = window.innerWidth;
        HEIGHT = window.innerHeight;
        div.style.width = WIDTH + "px";
        div.style.height = HEIGHT + "px";
      };

Минимальный набор JS для Web API:
Основы (ES6+):
Объявление переменных: const (основная), let (если нужно менять значение).
Стрелочные функции: () => {}.
Шаблонные строки: `Строка ${variable}`.
Деструктуризация массивов и объектов: { name } = user.
Работа с данными:
Типы данных: Строки, числа, булевы значения, массивы [] и объекты {}.
JSON: методы JSON.stringify() и JSON.parse() (критично для API).
Методы массивов: .map(), .filter(), .forEach().
Асинхронность :
Promise: понимание then()/catch().
async/await: современный способ работы с запросами.
fetch(): отправка и получение HTTP-запросов.
DOM (для отображения результатов API):
Поиск элементов: document.querySelector(), document.querySelectorAll().
Изменение контента: .textContent, .innerHTML.
Обработка событий: .addEventListener('click', ...).
HTML атрибут onclick

Итоговый «минимум»: научиться отправлять запрос через fetch, получать JSON, обрабатывать ошибки try/catch и выводить данные на страницу через DOM-методы.
